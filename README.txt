  
Scripts in scripts/ can be run via SUBMIT.sh

params/:	json files with model paramters
features/:	RData files with feature matrices (rows=sample, columns=covariate)
scripts/:	R-scripts to run
submit_job/:	bash script for cluster computing of scripts in scripts/
R/:		R codes for sourcing
dream_data/:	original dream data that is provided, do not write in here
data/: 		additional databases
scratch/:	folder for scratching
